Альтернативы с таким же способом ввода

Горячая клавиша → говорите → текст в активном поле
🍎
macOS — Встроенная диктовка
Бесплатно, ничего не нужно устанавливать
Бесплатно macOS
▼

Встроена в macOS 10.13+. Курсор в поле → горячая клавиша → говорите. Качество ниже AquaVoice, но без регистрации.

  • Включить: Системные настройки → Клавиатура → Диктовка → Вкл
  • Горячая клавиша: Fn Fn (двойное нажатие)
  • Хорошо для: быстрого теста без установки
🪟
Windows — Встроенный голосовой ввод
Бесплатно, работает из коробки
Бесплатно Windows 10/11
▼

Встроен в Windows 10 (1903+) и Windows 11. Поддерживает русский язык. Тот же принцип: курсор в поле → Win+H → говорите.

  • Горячая клавиша: Win + H
  • Закрыть панель: снова Win + H
  • Хорошо для: попробовать голосовой ввод без установки
🔠
Voice In — расширение Chrome
Тот же способ ввода, но только внутри браузера Chrome
Бесплатно Windows macOS
▼

Расширение добавляет голосовой ввод в любое поле на любом сайте в Chrome. Отлично работает в Claude, ChatGPT, Notion, Gmail.

  • Установка: Chrome Web Store → «Voice In Speech-to-Text»
  • Горячая клавиша: настраивается в расширении
  • Минус: работает только в Chrome, использует Google Speech API
💡 Когда переходить на платное

Если пользуетесь больше 30–60 минут в день — платные инструменты окупаются за счёт безлимита, более высокого качества и системной интеграции.

🖊
Superwhisper
Тот же способ, Whisper локально — данные не уходят в облако
14 дней бесплатно ~/мес macOS
▼

Горячая клавиша → говорите → текст в любом поле. Работает локально на Whisper — конфиденциально. Нет контекста экрана в отличие от AquaVoice.

  • Сайт: superwhisper.com
  • Плюс: глубокая настройка моделей и промптов постобработки
  • Минус: только Mac
🌊
Wispr Flow
AI-обработка, адаптирует стиль, есть Windows и iOS
14 дней бесплатно 2/мес macOS Windows iOS
▼

Тот же принцип работы, плюс AI-постобработка: фильтрует «эм», «ну», расставляет пунктуацию, адаптирует стиль под приложение (Slack, email, Notion — разный тон).

  • Сайт: wisprflow.ai
  • Плюс: кроссплатформенный — Mac, Windows, iOS
  • Плюс: Command Mode — редактирование текста голосом
  • Бесплатно: Basic — 2 000 слов/неделю после триала

🎯 Готово — что дальше

  • Первые 5 минут кажется странным — через 15 минут станет удобно
  • Начните с коротких фраз, постепенно переходите к длинным промптам

  • Если что-то не понравилось — попробуйте альтернативу выше, способ ввода тот же
